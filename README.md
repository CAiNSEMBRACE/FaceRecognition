# FaceRecognition
<h1>小学期Demo（全是bug，别看了）</h1>
<h1>主要功能：人脸识别打卡计时<h1>
  <p>环境：win10、vs2019、qt5.13、cmake、OpenCV-4.1.0、opencv_contrib-4.1.0（人脸识别模块）</p>
  <h2>三个模块<h2>
    <h3>1、信息采集：<h3>
      <p>获取摄像头图片裁剪头部，学号，姓名，班级并存入数据库（qt自带简单数据库）。</p>
    <h3>2、识别打卡：<h3>
      <p>面部识别打卡计时。</p>
    <h3>3、数据展示：<h3>
      <p>展示所有成员的学号、姓名、班级、学习时间。</p>
